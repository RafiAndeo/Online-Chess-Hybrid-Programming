Online Chess Hybrid Programming

How to play:
1. User menginstal python terlebih dahulu
2. User mengimport beberapa library yang digunakan pada program tersebut
3. User menginstal Radmin VPN dan menjalankan aplikasi tersebut
4. User membuat server baru pada Radmin VPN dengan cara mengisi nama server dan password server
5. User mengcopy ip address pada Radmin VPN dan mengubah host program menjadi ip address tersebut
6. User membagikan server yang sudah dibuat kepada user lain berupa nama server dan password server
7. User dengan ip address server akan menjalankan program server.py
8. User yang dibagikan nama dan password server hanya perlu menjalankan app.py saja
9. User yang menjalankan server.py dapat bermain juga dengan menjalankan app.py asalkan user yang terlibat hanya 2 user saja
10. User dapat membagikan nama server dan password server untuk mengajak user lain bermain
